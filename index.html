Proč se naučit Python?

  • Jednoduchá syntaxe - používá běžný anglický jazyk
  • Čitatelnost - kód je snadno pochopitelný
  • Univerzálnost - používá se v mnoha průmyslech
  • Velká komunita - mnoho knihoven a podpory

Proměnné

Proměnná je jako kontejner, který uchovává hodnotu. V Pythonu nemusíte deklarovat typ proměnné - Python ho určí automaticky.

Vytváření proměnných

Příklad:

jméno = "Jan"
věk = 25
výška = 1.75
je_student = True

Pravidla pro pojmenování

  • Název musí začínat písmenem nebo podtržítkem (_)
  • Může obsahovat písmena, čísla a podtržítka
  • Je rozlišován velký/malý znak (věk ≠ Věk)
  • Nepoužívejte Python klíčová slova (if, while, for, atd.)

Přiřazení více proměnných

x = y = z = 0 # Všechny proměnné mají hodnotu 0
a, b, c = 1, 2, 3 # Rozpacking hodnot

Typy Dat

Python má několik základních datových typů. Zde jsou ty nejdůležitější:

String (Text)

text = "Ahoj světe"
věta = 'Jednoduché i dvojité uvozovky"
dlouhý_text = """Více řádků
textu"""

Integer (Celé číslo)

číslo = 42
záporné = -10
nula = 0

Float (Desetinné číslo)

pi = 3.14
teplota = -5.5
vědecký_zápis = 1.5e-3

Boolean (Pravda/Nepravda)

je_pravda = True
je_nepravda = False

List (Seznam)

ovoce = ["jablko", "banán", "jahoda"]
čísla = [1, 2, 3, 4, 5]
smíšené = [1, "dva", 3.0, True]

Dictionary (Slovník)

osoba = {
    "jméno": "Jan",
    "věk": 25,
    "město": "Praha"
}

Operátory

Operátory se používají k provádění operací na proměnných a hodnotách.

Aritmetické operátory

sčítání = 5 + 3 # 8
odčítání = 10 - 4 # 6
násobení = 4 * 5 # 20
dělení = 15 / 3 # 5.0
celočíselné_dělení = 15 // 4 # 3
modulo = 17 % 5 # 2 (zbytek)
umocnění = 2 ** 3 # 8

Porovnávací operátory

5 == 5 # True (rovno)
5 != 3 # True (nerovno)
5 > 3 # True (větší než)
5 < 10 # True (menší než)
5 >= 5 # True (větší nebo rovno)
5 <= 3 # False (menší nebo rovno)

Logické operátory

True and False # False
True or False # True
not True # False

Podmínky (if/elif/else)

Podmínky vám umožňují spuštění různých kódů na základě různých podmínek.

Základní struktura

věk = 20

if věk >= 18:
    print("Jste dospělý")
else:
    print("Jste nezletilý")

Podmínky s elif

skóre = 75

if skóre >= 90:
    známka = "A"
elif skóre >= 80:
    známka = "B"
elif skóre >= 70:
    známka = "C"
else:
    známka = "F"

Vnořené podmínky

věk = 25
má_řidičský_průkaz = True

if věk >= 18:
    if má_řidičský_průkaz:
        print("Můžete řídit")
    else:
        print("Nemůžete řídit")
else:
    print("Jste příliš mladý")

Smyčky

Smyčky vám umožňují opakovat kód určitý počet krát.

For smyčka

for i in range(5):
    print(i) # Vypíše 0, 1, 2, 3, 4

ovoce = ["jablko", "banán", "jahoda"]
for kus in ovoce:
    print(kus)

While smyčka

i = 0
while i < 5:
    print(i)
    i += 1

Break a Continue

# Break - opuštění smyčky
for i in range(10):
    if i == 5:
        break
    print(i)

# Continue - přeskočení iterace
for i in range(5):
    if i == 2:
        continue
    print(i)

Funkce

Funkce je blok kódu, který provádí specifickou úlohu. Funkcí můžete použít vícekrát a činí код více organizovaný.

Vytváření funkce

def pozdrav(jméno):
    print(f"Ahoj, {jméno}!")

pozdrav("Jan") # Volání funkce

Funkce s návratovou hodnotou

def sčítej(a, b):
    return a + b

výsledek = sčítej(5, 3)
print(výsledek) # 8

Výchozí parametry

def pozdrav(jméno, lang="CS"):
    if lang == "CS":
        return f"Ahoj, {jméno}!"
    elif lang == "EN":
        return f"Hello, {jméno}!"

print(pozdrav("Jan")) # Ahoj, Jan!
print(pozdrav("John", "EN")) # Hello, John!

Více návratových hodnot

def dělení(a, b):
    podíl = a // b
    zbytek = a % b
    return podíl, zbytek

p, z = dělení(17, 5)
print(p, z) # 3 2
